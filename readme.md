# Defence & Border Surveillance Decision Dashboard
 
> **Academic Project** — Web Programming Subject  
> Semester: [4th semester] | Year: 2025–2026 | College: [VIT CHENNAI]
 
## Overview
 
A web-based decision support dashboard for defence and border surveillance operations. Built as part of the Web Programming coursework, this project demonstrates front-end development skills including responsive layout, interactive data tables, real-time UI elements, and map integration.
 
## Features
 
- **Command Dashboard** — Live stats, alert feed, and interactive surveillance map
- **Threat Assessment** — Filterable table of threats sorted by severity (High / Medium / Low)
- **Patrol Planning** — Shift schedules and patrol route management with status tracking
- **Live Clock** — Real-time clock in the header
- **Responsive Design** — Works on desktop and mobile
 
## Tech Stack
